## 基本的字体知识

### 命名规则

**个人建议 使用 `${名字}-${重量}-${W3C英文}`来进行命名，有利于排序！**

版本数据来自读取字体文件，读取软件：https://github.com/fontforge/fontforge

| 等级 | 重量 |          W3C英文          |                       中文 | 建议命名        |

| W1   | 100  |           Thin            |                 超细、极细 | W100-Thin       |
| W2   | 200  | Extra Light / Ultra Light |           瘦体、纤细、特细 | W200-ExtraLight |

### 字体风格

**斜体：`Oblique`、`Italic`** 

**宽度：`Condensed` 窄字、`Normal` 正常、`Expanded` 加大** 



### 字体子集（字体版本的英文）

关于字体的版本（思源黑体举例）

- **CN**：地区特定子集。更适合中国地区（中国大陆的汉字写法）。 还有HK、TW、JP、KR

- **SC** = Simplified Chines :  简体 大陆等

- **TC** = Traditional Chinese : 繁体 港台等



### 字体路径

**Windows10**

- `C:\Users\${USERNAME}\AppData\Local\Microsoft\Windows\Fonts`
- `C:\Windows\Fonts`
