## README

本人从事平面设计工作，本仓库仅备份我个人使用的字体（看得过眼且可直接商用，无须书面授权的中文字体）

**重复**或类似字体将选**其一**收录备份，信息资源均来自网络，如有版权争议与错误请及时提交 [issues](https://github.com/Tamshen/Freecommercialfont/issues)
